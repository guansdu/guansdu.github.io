打开Git bash

设置你的邮箱和用户名（我这里就是用的GitHub的邮箱，用户名随意）

生成新的SSH key

Hexo常用操作：

  • 生成静态页面：hexo g
  • 本地启动：hexo s(默认是localhost:4000，可在浏览器预览发布后的效果，若端口4000出错，可以尝试hexo s -p 3600修改到任意端口)
  • 发布到GitHub：hexo d
  • 新建文章： hexo new "blog-name"
