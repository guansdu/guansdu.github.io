关于C++中const的用法

Posted on 2016-07-26 Edited on 2017-09-13 In C++

被const关键字修饰的对象不能被修改，用于把一个对象变成常量。和#define不同的地方在于，const常量有数据类型，而宏常量没有，编译器对const常量可以进行类型安全检查，而对宏常量只进行字符替换。

const修饰普通变量

const int val和int const val这两种写法都可以，都表示val一旦初始化后值不可以被修改。

const修饰指针

const和指针在一起使用时，主要有下面几种写法：

  • (1) const int* p1：*p1不可变，p1可变；
  • (2) int* const p2：p2不可变，*p2可变；
  • (3) int const *p3：同(1)；
  • (4) const int* const p4：*p4不可变，p4不可变
